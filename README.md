Style guide and brand identity for Elements

https://elements-style-guide.netlify.app

## Initial Setup

```
npm install -g yarn
yarn install
```

## Dev

```
yarn run catalog-start
```

## Build

```
yarn run catalog-build
```

Output goes to `/catalog/build`
